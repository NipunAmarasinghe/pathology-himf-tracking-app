# Project Overview

This repository documents a Power Apps canvas application used to digitise pathology reagent intake and inventory tracking using HIMF identifiers.

## Objectives
- Standardise reagent intake data capture
- Improve traceability (HIMF, lot, supplier, expiry)
- Reduce manual errors and reagent wastage
- Enable audit-friendly reporting and future Power BI integration

## Scope
In scope:
- Barcode-assisted intake workflow
- SharePoint-backed inventory list
- Expiry and status logic
- Site/lab filtering for scalability

Out of scope (current version):
- Automated notifications via Power Automate
- Azure SQL migration
- OCR capture
