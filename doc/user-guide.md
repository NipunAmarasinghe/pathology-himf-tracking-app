# User Guide

## Reagent Intake Workflow
1. Open the app (Teams or web).
2. Select lab/site (if applicable).
3. Scan barcode or search item.
4. Confirm/enter:
   - HIMF
   - Lot number
   - Supplier
   - Received date
   - Expiry date
5. Save record.

## Change Item / Reset
Use the Change Item control to clear current selection and start a new intake entry.

## Common Errors
- Missing expiry date: record cannot be saved
- Invalid lot format: user prompted to correct input
- Unrecognised barcode: fallback to manual item search
