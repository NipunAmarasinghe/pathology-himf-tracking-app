# Deployment Notes

## Access
- Primary: Microsoft Teams
- Secondary: Web browser
- Optional: Managed iPad deployment (subject to organisation policy)

## Permissions
SharePoint list permissions should be role-based:
- Read/Write: lab staff
- Read-only: auditors/management
- Admin: app owners / pathology ICT
